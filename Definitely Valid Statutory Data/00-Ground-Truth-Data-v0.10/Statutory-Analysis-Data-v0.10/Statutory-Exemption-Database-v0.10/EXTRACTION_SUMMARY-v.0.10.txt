================================================================================
CSV FILE STRUCTURE
================================================================================

Columns (11 total):
  1. jurisdiction_name - Full name of jurisdiction (e.g., "Wyoming", "Federal")
  2. jurisdiction_code - Standard code (e.g., "US-WY", "US-FED")
  3. law_name - Name of the transparency law
  4. statute_citation - Primary statute citation
  5. section_reference - Specific statute sections referenced
  6. category - Main category (e.g., "Response Timeline", "Statutory Exemption")
  7. subcategory - Specific subcategory or provision type
  8. description - Detailed description of the right or exemption
  9. conditions - Conditions, limitations, or requirements
  10. notes - Additional context and extraction source
  11. extraction_date - ISO 8601 timestamp of extraction

  - This is due to varied formatting in source documents
  - Section references within documents were successfully extracted

Exemption Extraction:

Data Quality:
